# –讲述自己学习Markdown的过程、经历和实践，以及成果展示(各种Markdown语法的使用)
## 1.标题的展示
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
## 2.字体的变化
 **这是加粗的文字**  
 *这是斜体的文字*  
 ***这是斜体加粗的文字***  
 ~~这是加删除线的文字~~  
## 3.引用的方法
>这是引用的内容
>>这是引用的内容
>>>这是引用的内容
>>>>>>>>这是引用的内容
## 4.分割线的表示方法
三个或者三个以上的 - 或者 * 都可以。
---
----
***
*****
## 5.图片的引用
![图片alt](图片地址 ''图片title'')

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加
