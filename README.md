# –讲述自己学习Markdown的过程、经历和实践，以及成果展示(各种Markdown语法的使用)
## 1.标题的展示
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
## 2.字体的变化
 - **这是加粗的文字**  
 - *这是斜体的文字*  
 - ***这是斜体加粗的文字***  
 - ~~这是加删除线的文字~~  
## 3.引用的方法
>这是引用的内容
>>这是引用的内容
>>>这是引用的内容
>>>>>>>>这是引用的内容
## 4.分割线的表示方法
三个或者三个以上的 - 或者 * 都可以。
---
----
***
*****

## 6.超链接的使用
[超链接名](超链接地址 "超链接title")
title可加可不加
### 例如：
- [谷歌](http://google.com)
- [百度](http://baidu.com)
## 7.表格的使用
表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略
## 8.代码写入
语法：
单行代码：代码之间分别用一个反引号包起来
- `create database hero;`
- (```)
    function fun(){
         echo "这是一句非常牛逼的代码";
    }
    fun();
(```)
## 9.列表
- 列表内容
+ 列表内容
* 列表内容        

注意：- + * 跟内容之间都要有一个空格
